# pi-config
Config for the pi.dev

## Useful extensions

- [PI-superpowers](https://github.com/coctostan/pi-superpowers) - Brainstorming → Planning → TDD → Debugging → Code Review → Finishing — as composable skills your coding agent loads on demand.

How to install?

```javascript
pi install npm:pi-superpowers
```

How to uninstall?

```javascript
pi uninstall npm:pi-superpowers
```

- [PI-superpowers-plus](https://github.com/coctostan/pi-superpowers-plus) - Your coding agent doesn't just know the rules - it follows them. Skills teach the agent what to do (brainstorm before building, write tests before code, verify before claiming done). Extensions enforce it in real time (the workflow monitor watches every file write and warns when you skip the test).

How to install?

```javascript
pi install npm:pi-superpowers-plus
```

How to uninstall?

```javascript
pi uninstall npm:pi-superpowers-plus
```
